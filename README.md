# IT Foundations Portfolio

Welcome to my **IT Foundations Portfolio** — a collection of hands-on projects that demonstrate my skills in system administration, networking, security, automation, and virtual lab setup.

This repository showcases practical work completed to build a strong foundation in core IT concepts and real-world troubleshooting. Each project reflects a different area of learning and practice, from Active Directory deployment to home network design and system hardening.

---

## Portfolio Projects

### 1. Active Directory Deployment
Built and configured an Active Directory environment to practice domain management, user and group administration, and basic enterprise identity services.

### 2. Backup Automation
Created an automated backup solution to improve data protection, consistency, and recovery readiness.

### 3. Baseline System Hardening
Applied security best practices to strengthen a system’s defenses by reducing unnecessary exposure and improving configuration hygiene.

### 4. Home Network Mapping
Designed and documented a home network layout to better understand device connectivity, IP relationships, and network structure.

### 5. Home Virtual Lab Setup
Set up a virtual lab environment for safe testing, hands-on learning, and practicing IT tasks in an isolated environment.
